# Telegram Bot Chat ID Finder

A simple web page to help you discover your Telegram bot chat ID.

## How to Use

1. Add the bot to a Telegram group
2. Send a message in that group
3. The bot will respond with the chat ID

## Features

- Simple and clean interface
- Displays the chat ID prominently
- Shows error messages if something goes wrong
- Instructions for getting your chat ID

## Technical Details

This is a static HTML page that can be hosted on GitHub Pages. The page includes:

- A title: "discover telegram bot chat id"
- A description in thin font explaining how to get the chat ID
- A prominent display area for the chat ID
- Error handling
- Instructions section

## Hosting

This page is designed to be hosted on GitHub Pages at: https://lalithchary.github.io/

## Security

Your Telegram bot chat ID is not stored or transmitted anywhere. This is a client-side only application.
